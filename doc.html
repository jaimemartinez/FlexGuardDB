Authentication

Register User

POST /register

Registers a new user with email, username, and a secure password.

Password Requirements: Minimum 8 characters, at least one uppercase letter, one lowercase letter, one number, and one special character. 

Request Body:
{
    "email": "user@example.com",
    "username": "testuser",
    "password": "Password@123"
}

Response: 201 Created - User registered successfully.

Data Types

When creating a table, you can specify the following data types for the columns:

  • INTEGER: Stores integer values.
  • STRING: Stores text values.
  • BOOLEAN: Stores true/false values.
  • DATE: Stores date objects.
  • NULL: Allows storing null values.

Table Operations

Create Table

POST /createTable

Creates a new table with the specified columns.

Request Body:
{
    "tableName": "users",
    "columns": [
        { "name": "id", "type": "INTEGER", "primaryKey": true },
        { "name": "name", "type": "STRING" },
        { "name": "age", "type": "INTEGER" }
    ]
}

Response: 201 Created - Table created successfully.

Create Table If Not Exist

POST /createTableIfNotExist

Creates a table if it doesn't already exist.

Request Body:
{
    "tableName": "users",
    "columns": [
        { "name": "id", "type": "INTEGER", "primaryKey": true },
        { "name": "name", "type": "STRING" },
        { "name": "age", "type": "INTEGER" }
    ]
}

Response: 201 Created or 200 OK if the table already exists.

Insert Data

POST /insert

Inserts a new row into the specified table.

Request Body:
{
    "tableName": "users",
    "values": [1, "Alice", 30]
}

Response: 201 Created - Data inserted successfully.

Select Data

GET /select

Selects data from the specified table.

Query Parameters:
?tableName=users&column=age&operator=%3E&value=25

Response: Selected rows in JSON format.

Update Data

POST /update

Updates rows that match the given conditions.

Request Body:
{
    "tableName": "users",
    "conditions": [{ "column": "name", "operator": "==", "value": "Alice" }],
    "updates": { "age": 31 }
}

Response: Number of rows updated.

Get Table Names

GET /tableNames

Retrieves the names of all available tables.

Response: Array of table names in JSON format.

Get Table Schema

GET /tableSchema

Retrieves the schema of the specified table.

Query Parameter:
?tableName=users

Response: JSON object representing the table schema.
